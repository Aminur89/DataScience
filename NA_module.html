Numerical Analysis

Key aspects of mathematics and statistics for data science and artificial intelligence.

Learning Objectives

  • Develop a systematic understanding of foundational mathematical principles and methods, as well as core and specialised concepts underpinning computing logic.
  • Understand the foundation for the development and application of programming and data-driven techniques, from both a theoretical and practical viewpoint.
  • Facilitate the ability to interpret the results generated when using these data science and ai tools.
  • Gain an understanding of the real-world applications of these computational tools, and contemporary issues related to these computational techniques.
  • The opportunity to take a reflective and independent approach to the learning process.

Unit 1: Types and Sources of Data and an Introduction to R

Unit 1 provided a critical overview of statistics' importance across fields and introduced the essentials of R for data analysis. It highlighted the dual nature of statistics as both enlightening and potentially misleading.

    Key Insights

  • Critical Perspective on Statistics: The unit underscored the importance of scrutinizing statistical data, urging a deeper look into its sources and implications.
  • R Programming Fundamentals: Diving into R and RStudio was challenging but rewarding, offering practical tools for data manipulation and analysis.

    Practical Applications

  • Measurement Scales in R: Understanding and applying different measurement scales was invaluable, laying the groundwork for accurate data interpretation.
  • Descriptive Statistics: The exercises in R enhanced my ability to summarize and analyze data, a skill vital for any data-driven decision-making process.

The unit was an eye-opener, emphasizing the critical role of statistics in various domains and equipping me with foundational R programming skills. It has sparked a keen interest in further exploring statistical analysis and its applications.

Unit 2: Data Structures in R

This week, I learned about various data structures in R, including vectors, matrices, and the introduction to data frames. Understanding data structures is crucial for working effectively with any software, as they define how data is stored, interpreted, and manipulated.

    Key Learnings:

  • Vectors: I grasped that vectors are one-dimensional arrays that can hold elements of the same data type (numeric, character, etc.). They are a fundamental building block for storing and working with simple data lists in R. I practiced creating and manipulating vectors using functions like c(), length(), and indexing.
  • Matrices: Matrices extend the concept of vectors to two dimensions, allowing storage of data in rows and columns. They can also hold elements of the same data type. I learned how to create matrices using the matrix() function and explored basic operations like matrix transposition.

    Challenges and Wins:

  • Initially, I found it slightly challenging to differentiate between vectors and matrices conceptually. However, working through examples and exercises helped solidify the differences. Practicing creating matrices with different dimensions and performing operations like subsetting rows and columns was particularly helpful.

Unit 3: Data Management in R

Week 3's focus on data management in R was a deep dive into the essential steps of preparing data for analysis. Learning to read various data formats into RStudio, examining datasets, and managing variable types has significantly bolstered my understanding of R's capabilities for data analysis.

    Key Learnings:

  • I practiced reading various data formats like CSV and Excel files using functions like read.csv() and readxl::read_excel().
  • I learned to examine entire datasets using summary() to get an overview of data types and basic statistics. The str() function provided detailed information about the data structure and variable types.
  • To explore specific variables, I practiced using functions like str(), summary(), and table() to understand data types, value distribution, and identify potential outliers.
  • I grasped how to convert variable types using functions like as.numeric() and as.character() for effective data manipulation and analysis.
  • The unit introduced different approaches to handle missing values. I learned about removing rows/columns with missing data, using imputation techniques to fill in missing values, and employing functions like na.rm to exclude missing values during calculations.

    Challenges and Wins:

    Initially, understanding the nuances of different data types and interpreting summary statistics proved slightly challenging. However, working through examples that showcased real-world data scenarios helped solidify these concepts. Practicing data cleaning tasks, like converting variable types and handling missing values, was particularly rewarding as I saw the data transform into a usable format for analysis.
