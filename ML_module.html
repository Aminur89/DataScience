Machine Learning

In the Machine Learning module, we will develop a solid understanding of machine learning techniques, with a focus on real-world data analytics applications. Through a blend of theoretical concepts and hands-on experience, we will explore key machine learning paradigms, models, and algorithms using modern programming tools like Python and R.

Learning Objectives

  • Develop a strong conceptual understanding of key machine learning paradigms, algorithms, and frameworks.
  • Apply mathematical concepts and software tools (Python, R) to implement machine learning techniques in real-world data analytics projects.
  • Critically evaluate machine learning models and their applicability to various datasets, considering potential risks and uncertainties.
  • Explore the legal, social, ethical, and professional challenges associated with the deployment of machine learning systems.
  • Build the skills needed to effectively collaborate in a virtual professional team environment, addressing real-life perspectives on team roles and organization.
  • Gain exposure to the latest developments in machine learning model deployment and state-of-the-art techniques.

Unit 1: Introduction to Machine Learning (ML)

In this unit, I participated in a thoughtful discussion about the impact of Industry 4.0 on the healthcare sector, specifically focusing on the growing reliance on information systems and the associated cybersecurity risks. I shared insights on the 2017 WannaCry ransomware attack on the UK's National Health Service (NHS), highlighting the chaos caused by system failures and the critical need for strong security measures.

The discussion helped me engage with peers on the importance of balancing technological innovation with the need to protect sensitive patient data. It also emphasized the role of cybersecurity in maintaining trust and continuity within healthcare systems as they undergo digital transformation.

Unit 2: Exploratory Data Analysis

In this unit, I learned the importance of Exploratory Data Analysis (EDA) in ensuring the accuracy and relevance of data for machine learning projects. EDA allows data scientists to validate datasets, identify anomalies, and gain valuable insights that might not have been initially obvious. I practiced performing EDA on the Auto-mpg dataset using tools like Jupyter Notebook and Google Colab, working through tasks such as identifying missing values, visualizing correlations with heat maps, and refining feature variables.

This unit reinforced the critical role EDA plays in preparing data for machine learning and highlighted the iterative nature of feature engineering. Additionally, I enhanced my skills in using visual analysis techniques to better understand datasets and ensure they are ready for machine learning models.

Unit 3:Correlation and Regression

    In this unit, I delved into correlation and regression, two essential statistical techniques that help quantify relationships between variables. I explored how correlation measures the strength and direction of relationships, while regression provides a way to predict outcomes based on these relationships.

    A key part of this unit involved hands-on activities using Jupyter Notebook and Google Colab. I worked with several programs, including covariance_pearson_correlation.ipynb, linear_regression.ipynb, multiple_linear_regression.ipynb, and polynomial_regression.ipynb. By changing variables and data points within these programs, I observed how these modifications impacted both correlation and regression outcomes. This practical approach allowed me to see firsthand the dynamic nature of data relationships and the importance of choosing the right model for analysis.

    These activities helped solidify my understanding of how to apply correlation and regression techniques to real-world datasets, an essential skill for any data scientist.

Unit 4: Linear Regression with Scikit-Learn

    In this unit, I explored linear regression using Scikit-Learn and practiced applying the fundamentals of regression modeling. I began by running the step-by-step demo correlation_regression_fuelconsumption.ipynb using the FuelConsumption dataset. This activity allowed me to directly connect the theoretical concepts from previous units with practical implementation in Jupyter Notebook and Google Colab. I experimented with data points to see how they influenced correlation and regression outcomes.

    I then tackled a more advanced task using the Global_Population.csv and Global_GDP.csv datasets. In Task A, I pre-processed the data to handle missing values and calculated the mean population and mean per capita GDP of each country (2001-2021). By investigating the correlation between these variables, I learned to interpret the generated plots and evaluated the Pearson Correlation Coefficient.

    In Task B, I performed linear regression with the mean population as the independent variable and the mean per capita GDP as the dependent variable. This hands-on experience deepened my understanding of linear regression's real-world applications, especially in socioeconomic analysis. I also had the chance to review and discuss my results with my team, reinforcing the importance of collaboration in data science projects.

    Through this unit, I honed my skills in regression modeling, data pre-processing, and the effective use of the Scikit-Learn library in Python.

Unit 5:Clustering

In this unit, I not only explored clustering techniques but also learned how to measure similarity between data points using the Jaccard coefficient. As a practical exercise, I calculated the Jaccard coefficient for three individuals based on their pathological test results. This exercise helped me understand how to quantify the similarity between different datasets, a skill crucial for clustering tasks.

The Jaccard coefficient calculations for pairs (Jack, Mary), (Jack, Jim), and (Jim, Mary) demonstrated how data features can be compared and grouped based on their similarities. This hands-on activity deepened my understanding of data analysis in real-world scenarios, emphasizing the importance of accurately evaluating clusters.

Moreover, this unit reinforced the idea that working with personal data in machine learning requires careful consideration of legal, social, and ethical implications, as maintaining data privacy and integrity is paramount.
