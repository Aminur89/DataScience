Deciphering Big Data

In the Machine Learning module, we will develop a solid understanding of machine learning techniques, with a focus on real-world data analytics applications. Through a blend of theoretical concepts and hands-on experience, we will explore key machine learning paradigms, models, and algorithms using modern programming tools like Python and R.

Learning Objectives

  • Develop a strong conceptual understanding of key machine learning paradigms, algorithms, and frameworks.
  • Apply mathematical concepts and software tools (Python, R) to implement machine learning techniques in real-world data analytics projects.
  • Critically evaluate machine learning models and their applicability to various datasets, considering potential risks and uncertainties.
  • Explore the legal, social, ethical, and professional challenges associated with the deployment of machine learning systems.
  • Build the skills needed to effectively collaborate in a virtual professional team environment, addressing real-life perspectives on team roles and organization.
  • Gain exposure to the latest developments in machine learning model deployment and state-of-the-art techniques.

Unit 1: Introduction to Machine Learning (ML)

In this unit, I participated in a thoughtful discussion about the impact of Industry 4.0 on the healthcare sector, specifically focusing on the growing reliance on information systems and the associated cybersecurity risks. I shared insights on the 2017 WannaCry ransomware attack on the UK's National Health Service (NHS), highlighting the chaos caused by system failures and the critical need for strong security measures.

The discussion helped me engage with peers on the importance of balancing technological innovation with the need to protect sensitive patient data. It also emphasized the role of cybersecurity in maintaining trust and continuity within healthcare systems as they undergo digital transformation.

Unit 2: Exploratory Data Analysis

In this unit, I learned the importance of Exploratory Data Analysis (EDA) in ensuring the accuracy and relevance of data for machine learning projects. EDA allows data scientists to validate datasets, identify anomalies, and gain valuable insights that might not have been initially obvious. I practiced performing EDA on the Auto-mpg dataset using tools like Jupyter Notebook and Google Colab, working through tasks such as identifying missing values, visualizing correlations with heat maps, and refining feature variables.

This unit reinforced the critical role EDA plays in preparing data for machine learning and highlighted the iterative nature of feature engineering. Additionally, I enhanced my skills in using visual analysis techniques to better understand datasets and ensure they are ready for machine learning models.
